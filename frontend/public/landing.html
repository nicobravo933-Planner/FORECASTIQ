Capabilities

Qué hace ForecastIQ

Selección automática de modelo

Un score de calidad evalúa el dataset (0–100). Score < 40 → solo Moving Average. Score ≥ 75 → LightGBM disponible, incluyendo features de calendario como variables exógenas.

MA · Holt-Winters · SARIMA · LightGBM
Eventos como features exógenas

Feriados nacionales y eventos comerciales argentinos (Hot Sale, Cyber Monday, Día del Padre) se inyectan como variables exógenas en LightGBM. El modelo aprende el impacto real de cada evento en tus ventas.

feature engineering · calendario AR · exógenas
Chat IA sobre tus datos

El LLM recibe el contexto del último forecast como system prompt: métricas, modelo elegido, columnas detectadas, horizonte. Preguntás en lenguaje natural y responde sobre tu serie temporal específica.

Claude · context injection · RAG lite
Métricas de evaluación correctas

Reportamos WAPE (Weighted Absolute Percentage Error) y Forecast Value Added respecto al benchmark naïve. Intervalos de confianza al 90% por bootstrap o distribución del error histórico.

WAPE · FVA · IC 90% · naïve benchmark
Pipeline

Cómo funciona

PASO 01
Subís tu CSV

Cargás tu serie temporal. El sistema detecta automáticamente las columnas de fecha y valor, evalúa la calidad del dataset y genera estadísticas descriptivas.

PASO 02
ETL + selección de modelo

Winsorización de outliers, imputación de valores faltantes y detección de periodicidad. El score de calidad determina qué modelos ML están disponibles para tu dataset.

PASO 03
Forecast + métricas

Obtenés el forecast con WAPE, FVA e intervalos de confianza al 90%. El resultado queda guardado en tu historial para comparar runs futuros.
